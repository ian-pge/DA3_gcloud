# Depth Anything 3: Feed-Forward Gaussian Splatting

This project runs feed-forward Gaussian Splatting inference using the **Depth Anything 3 (Giant)** model. It takes standard 2D images and converts them into 3D Gaussian Splat `.ply` files.
